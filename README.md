# 1️⃣ INTRODUCTION & MS OFFICE OVERVIEW

## 1. Data Processing ⭐
Converting raw data into meaningful information through operations (input → process → output → storage).

## 2. MS Office ⭐
Suite of productivity applications by **Microsoft**.
- **MS Word** — word processing
- **MS Excel** — spreadsheet
- **MS PowerPoint** — presentations
- **MS Access** — database
- **MS Outlook** — email
- **MS OneNote** — notes
- **MS Publisher** — publishing

## 3. Application Software Types
| Software | Use | Example |
|----------|-----|---------|
| Word Processor | Documents | MS Word |
| Spreadsheet | Calculations, data | MS Excel |
| Presentation | Slides | MS PowerPoint |
| Database | Data management | MS Access |

## 4. MS Office Versions
2003, 2007 (ribbon introduced), 2010, 2013, 2016, 2019, 2021, **Microsoft 365** (subscription).

## 5. Ribbon ⭐
The tabbed toolbar at the top (introduced in Office 2007). Contains tabs → groups → commands.

## 6. Common Office Components ⭐
- **Title Bar** — shows file name
- **Ribbon** — command tabs
- **Quick Access Toolbar** — frequently used commands
- **Status Bar** — bottom info bar
- **File/Backstage (Office button)** — file operations
- **Scroll bars**

## 7. File Menu / Backstage View
Contains: New, Open, Save, Save As, Print, Share, Export, Close.

## 8. Quick Access Toolbar (QAT)
Customizable toolbar with common commands (Save, Undo, Redo). Above or below ribbon.

## 9. Office File Formats ⭐⭐
| App | Old (97-2003) | New (2007+) |
|-----|---------------|-------------|
| Word | .doc | .docx |
| Excel | .xls | .xlsx |
| PowerPoint | .ppt | .pptx |
| Access | .mdb | .accdb |

New formats are XML-based (Open XML).

## 10. Cloud Integration
Microsoft 365 + OneDrive = cloud storage, collaboration, real-time co-authoring.

# 3️⃣ MS EXCEL — SPREADSHEET ⭐⭐ EXAM HEAVY

## 46. MS Excel ⭐
Spreadsheet application for calculations, data analysis, charts.

## 47. Excel File Extension ⭐
- **.xlsx** (2007+)
- **.xls** (97-2003)
- **.xlsm** (macro-enabled)
- **.csv** (comma-separated)

## 48. Excel Components ⭐⭐
- **Workbook** — the entire Excel file
- **Worksheet/Sheet** — single page (tabs at bottom)
- **Cell** — intersection of row and column
- **Cell Reference** — address (e.g., A1, B5)
- **Name Box** — shows active cell reference
- **Formula Bar** — shows/edits cell content

## 49. Rows and Columns ⭐⭐
- **Rows** — horizontal, numbered (1, 2, 3...)
- **Columns** — vertical, lettered (A, B, C...)
- **Cell** — intersection (A1 = column A, row 1)

### Excel limits (2007+)
- **Rows: 1,048,576** (2²⁰)
- **Columns: 16,384** (XFD)

## 50. Cell Reference Types ⭐⭐
| Type | Example | Behavior |
|------|---------|----------|
| **Relative** | A1 | Changes when copied |
| **Absolute** | $A$1 | Fixed when copied |
| **Mixed** | $A1 or A$1 | Partially fixed |

**F4** toggles reference types.

## 51. Formula vs Function ⭐
- **Formula**: user-created calculation (=A1+B1)
- **Function**: built-in predefined formula (=SUM(A1:A10))
- All formulas start with **=**

## 52. Basic Operators ⭐
- Arithmetic: + - * / ^ (power) %
- Comparison: = > < >= <= <>
- Order: PEMDAS/BODMAS

## 53. Common Functions ⭐⭐ MEMORIZE
| Function | Purpose |
|----------|---------|
| **SUM()** | Add values |
| **AVERAGE()** | Mean |
| **COUNT()** | Count numbers |
| **COUNTA()** | Count non-empty |
| **MAX()** | Largest |
| **MIN()** | Smallest |
| **IF()** | Conditional |
| **ROUND()** | Round number |
| **SQRT()** | Square root |
| **POWER()** | Exponent |

## 54. Statistical Functions ⭐
- **COUNT()** — count cells with numbers
- **COUNTA()** — count non-empty cells
- **COUNTBLANK()** — count empty cells
- **COUNTIF()** — count by condition
- **MEDIAN()**, **MODE()**, **STDEV()**

## 55. Logical Functions ⭐
- **IF(condition, true_val, false_val)**
- **AND()**, **OR()**, **NOT()**
- **IFERROR()**
- Nested IF

### IF Example
```
=IF(A1>=40, "Pass", "Fail")
```

## 56. Text Functions ⭐
| Function | Purpose |
|----------|---------|
| **LEN()** | Length of text |
| **LEFT()** | Left characters |
| **RIGHT()** | Right characters |
| **MID()** | Middle characters |
| **UPPER()** | Uppercase |
| **LOWER()** | Lowercase |
| **CONCATENATE()/CONCAT()** | Join text |
| **TRIM()** | Remove extra spaces |

## 57. Lookup Functions ⭐⭐
- **VLOOKUP()** — vertical lookup
- **HLOOKUP()** — horizontal lookup
- **INDEX()** and **MATCH()**
- **XLOOKUP()** (newer)

### VLOOKUP syntax
```
=VLOOKUP(lookup_value, table_array, col_index, [range_lookup])
```

## 58. Date & Time Functions ⭐
- **TODAY()** — current date
- **NOW()** — current date and time
- **DAY()**, **MONTH()**, **YEAR()**
- **DATE()**, **TIME()**

## 59. SUMIF and COUNTIF ⭐
- **SUMIF(range, criteria, sum_range)** — conditional sum
- **COUNTIF(range, criteria)** — conditional count
- **AVERAGEIF()**, **SUMIFS()**, **COUNTIFS()**

## 60. Cell Range ⭐
- Group of cells: **A1:A10** (colon = range)
- **A1:C5** = rectangular block
- Comma = separate cells: A1,B2,C3

## 61. AutoFill ⭐
Drag fill handle to copy/extend series (1,2,3... or Jan, Feb...). Fill handle = small square at cell corner.

## 62. Charts in Excel ⭐⭐
| Chart Type | Use |
|-----------|-----|
| **Column/Bar** | Compare values |
| **Line** | Trends over time |
| **Pie** | Parts of whole |
| **Scatter (XY)** | Correlation |
| **Area** | Cumulative trends |

Insert tab → Charts.

## 63. Sorting ⭐
Arrange data: ascending (A-Z, small-large) or descending. Data tab → Sort.

## 64. Filtering ⭐
Show only matching data. Data tab → Filter (AutoFilter). Advanced Filter for complex criteria.

## 65. Conditional Formatting ⭐
Format cells based on conditions (color scales, data bars, icon sets, highlight rules). Home tab.

## 66. Pivot Table ⭐⭐
Summarize and analyze large data. Drag fields to rows, columns, values. Insert tab → PivotTable.

## 67. Freeze Panes ⭐
Keep rows/columns visible while scrolling. View tab → Freeze Panes.

## 68. Cell Formatting ⭐
- Number formats: General, Number, Currency, Percentage, Date
- Borders, fill color
- Merge & Center
- Text alignment, wrap text

## 69. Data Validation ⭐
Restrict cell input (dropdown lists, number ranges, dates). Data tab → Data Validation.

## 70. Error Values ⭐⭐
| Error | Meaning |
|-------|---------|
| **#DIV/0!** | Division by zero |
| **#VALUE!** | Wrong data type |
| **#REF!** | Invalid cell reference |
| **#NAME?** | Unrecognized name/function |
| **#N/A** | Value not available |
| **#NUM!** | Invalid number |
| **#NULL!** | Invalid intersection |
| **####** | Column too narrow |

## 71. Absolute vs Relative Reference Example ⭐
- Copy `=A1+B1` down → `=A2+B2` (relative)
- Copy `=$A$1+B1` down → `=$A$1+B2` (A1 fixed)

## 72. Goal Seek
Find input value for desired result. Data tab → What-If Analysis → Goal Seek.

## 73. Macros in Excel
Automate repetitive tasks (VBA). Developer tab or View → Macros.

## 74. Worksheet Operations ⭐
- Insert/Delete sheet
- Rename sheet (double-click tab)
- Move/Copy sheet
- Color sheet tab

## 75. Cell Comments/Notes
Add notes to cells (Review tab). Red triangle indicates a note.

## 76. Print Area
Define what to print. Page Layout → Print Area.

## 77. Headers/Footers in Excel
Page Layout view or Insert → Header & Footer.

## 78. Wrap Text
Display long text in multiple lines within a cell. Home tab → Wrap Text.

## 79. Merge & Center ⭐
Combine cells and center content. Home tab.

## 80. Format Painter
Copy formatting from one cell to another (paintbrush icon).

## 81. Series and Patterns
AutoFill recognizes patterns: numbers, dates, days, months, custom lists.

## 82. Excel Keyboard Shortcuts ⭐
- **Ctrl+C/V/X** — copy/paste/cut
- **Ctrl+Z/Y** — undo/redo
- **Ctrl+Arrow** — jump to data edge
- **Ctrl+Home** — go to A1

## 84. Name Box ⭐
Left of formula bar. Shows active cell reference. Can name ranges.

## 85. Formula Bar ⭐

## 86. Active Cell
Currently selected cell (highlighted border). Reference shown in Name Box.

## 87. Worksheet vs Workbook ⭐⭐
- **Workbook** = entire Excel file (.xlsx)
- **Worksheet** = single sheet within workbook

## 88. Gridlines
Light lines separating cells. Can be hidden/shown (View tab).

## 89. Number Formatting ⭐
- Currency (₹, $)
- Percentage (%)
- Comma style (1,000)
- Decimal places
- Scientific

## 90. Hyperlinks in Excel
Ctrl+K. Link cells to webpages, files, other cells.

# 5️⃣ MS ACCESS — DATABASE ⭐⭐

## 116. MS Access ⭐
Database management software (RDBMS) by Microsoft. Part of MS Office.

## 117. Access File Extension ⭐
- **.accdb** (2007+)
- **.mdb** (2003 and earlier)

## 118. Access Objects ⭐⭐ MEMORIZE
| Object | Purpose |
|--------|---------|
| **Table** | Stores data (rows + columns) |
| **Query** | Retrieve/filter data |
| **Form** | User interface for data entry |
| **Report** | Formatted printed output |
| **Macro** | Automate tasks |
| **Module** | VBA code |

## 119. Table ⭐⭐
Stores data in rows (records) and columns (fields).
- **Field** = column (attribute)
- **Record** = row (tuple)

## 120. Field Data Types ⭐⭐
| Data Type | Use |
|-----------|-----|
| **Short Text** | Text up to 255 chars |
| **Long Text/Memo** | Large text |
| **Number** | Numeric |
| **Date/Time** | Dates |
| **Currency** | Money |
| **AutoNumber** | Auto-incrementing |
| **Yes/No** | Boolean |
| **OLE Object** | Images, files |
| **Hyperlink** | Links |
| **Attachment** | Attached files |

## 121. Primary Key in Access ⭐
Uniquely identifies each record. Often AutoNumber. Cannot be duplicate or null.

## 122. Query ⭐⭐
Retrieves specific data from tables.

### Query Types
- **Select Query** — retrieve data
- **Action Queries**: Update, Append, Delete, Make-Table
- **Parameter Query** — prompts for input
- **Crosstab Query** — summary

## 123. Query Design / QBE ⭐
**Query By Example** — visual query design (drag fields, set criteria). Or SQL view.

## 124. Form ⭐⭐
User-friendly interface for entering/viewing data. Better than entering directly in tables.

### Form Views
- Form View (use)
- Design View (build)
- Layout View

## 125. Report ⭐⭐
Formatted output for printing (summaries, grouped data, totals).

### Report Sections
- Report Header/Footer
- Page Header/Footer
- Detail (records)
- Group Header/Footer

## 126. Relationships in Access ⭐
Link tables via keys (one-to-one, one-to-many, many-to-many). Relationships window. Enforce referential integrity.

## 127. Macro in Access
Automate tasks without coding (open form, run query, etc.).

## 128. Module
VBA code for complex automation.

## 129. Wizards in Access
Step-by-step helpers: Table Wizard, Form Wizard, Report Wizard, Query Wizard.

## 130. Validation Rule
Restrict data entered in a field (e.g., >0, specific format).

## 131. Input Mask
Controls format of data entry (e.g., phone number, date format).

## 132. Default Value
Auto-filled value for a field if none entered.

## 133. Index in Access
Speeds up searching and sorting. Primary key auto-indexed.

## 134. Lookup Field
Field that gets values from another table or list (dropdown).

## 135. Field Properties
Size, format, caption, default value, validation rule, required, indexed.

## 136. Datasheet View vs Design View ⭐
- **Datasheet View** — view/enter data (like a table)
- **Design View** — define structure (fields, types, properties)

## 137. Sorting and Filtering in Access
Sort records (ascending/descending). Filter to show specific records.

## 138. Referential Integrity in Access ⭐
Ensures relationships valid (foreign key matches primary key). Prevents orphan records.

## 139. Importing/Exporting Data
Import from/export to Excel, CSV, text, other databases.

## 140. Access vs Excel ⭐
| Access | Excel |
|--------|-------|
| Database (relational) | Spreadsheet |
| Large structured data | Calculations, analysis |
| Tables, queries, forms, reports | Cells, formulas, charts |
| Multiple related tables | Single sheets |
| Better for data integrity | Better for quick calculations |
